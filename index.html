📦 出貨雙重確認

從常用商品挑選今日要出的

在右側輸入今天每個 SKU 要出貨的數量，留空或 0 = 今天不出。輸入完按下方按鈕進入備貨確認。

從 Excel 出貨總表貼上

手機 OneDrive 打開「出貨確認表」→「出貨總表」分頁 → 選 B 到 H 欄資料 → 複製 → 切回此頁長按下方框「貼上」

或上傳 CSV 檔案
支援的資料格式
每列一個商品，欄位用 Tab 或逗號分隔：
  • 2 欄：SKU + 數量
  • 3 欄：SKU + 名稱 + 數量
  • 7 欄（從 Excel 出貨總表直接貼）：自動辨識
數量 = 0 的列會自動跳過。
