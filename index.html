📖 Google Sheets 設定教學

1. 開新的 Google Sheets
2. 選單「擴充功能 → Apps Script」
3. 貼以下程式碼: 

function doPost(e) {
  const d = JSON.parse(e.postData.contents);
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  if (d.type === "picking_outbound") {
    sheet.appendRow([
      new Date(),
      d.doc_no, d.customer,
      d.items.map(i =>
        i.sku + "x" + i.qty
      ).join(", ")
    ]);
  }
  return ContentService
    .createTextOutput("OK");
}

4. 「部署 → 新增部署」, 類型「網頁應用程式」
5. 「執行身分: 我」, 「存取權: 任何人」
6. 複製 Web App 網址 → 貼上方

💾 資料搬家 / 備份

把整個 App 的資料 (商品庫 + 庫存 + 規格對應 + 出貨紀錄 + 設定) 存成一個 JSON 檔, 之後可以在另一個網址 / 手機匯入。

⚠ 瀏覽器資料依「網址」分開。例如 koca-picking.pages.dev 跟 xxx.workers.dev 是兩個獨立的資料盒, 互相看不到。換網址前先匯出, 在新網址匯入, 才能延續。

包含哪些資料?
  • 商品庫 (SKU + 名稱 + 分類 + 條碼 + 必要商品標記)
  • 倉庫清冊 (75 SKU 的現有庫存)
  • 規格 → SKU 對應 (蝦皮揀貨單用)
  • 出貨紀錄 (一般 + 補寄)
  • 設定 (聲音 / Webhook / 主題)
  • 當前未完成的備貨清單 (state)
