📖 Google Sheets 設定教學

1. 開新的 Google Sheets
2. 選單「擴充功能 → Apps Script」
3. 貼以下程式碼:

function doPost(e) {
  const d = JSON.parse(e.postData.contents);
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  // 主表: 每筆出貨一列
  if (d.type === "shipping_complete") {
    sheet.appendRow([
      new Date(d.completedAt),
      d.platform,
      d.summary.totalSku,
      d.summary.totalExpected,
      d.summary.totalPacked,
      d.summary.completionRate + "%",
      d.summary.parcelCount,
      d.items.map(i =>
        i.sku + "x" + i.packed
      ).join(", ")
    ]);
  }
  return ContentService
    .createTextOutput("OK");
}

4. 「部署 → 新增部署」, 類型「網頁應用程式」
5. 「執行身分: 我」, 「存取權: 任何人」
6. 部署 → 複製 Web App 網址 → 貼到上方
